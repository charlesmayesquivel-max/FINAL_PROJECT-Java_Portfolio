# DCIT 50 – Object-Oriented Programming

## Repository Overview

This repository serves as a compilation of all academic activities, exercises, and requirements completed for **DCIT 50 – Object-Oriented Programming** during the **First Semester of Academic Year 2025–2026**.

The contents of this repository reflect the application of fundamental object-oriented programming concepts, including but not limited to abstraction, encapsulation, inheritance, and polymorphism, as covered throughout the course.
